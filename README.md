# Haskell-Cheatsheet

Arithmetic

2+15
> 17

2-15
> -13

2+(-15)
> -13

49*100
>4900

5/2
> 2.5

div 5 2
> 2

5 `div` 2
> 2

2+4*3
> 14

Only div 5 2 and 5 `div` 2 act like integer arithmetic in other languages.

Watch out for

5* -3  won't work, must do 5*(-3)

Boolean:
&& and
|| or

True && True
> True

True && False
> False

True || True
> True

True || False
> True

Interesting:
not False
> True
